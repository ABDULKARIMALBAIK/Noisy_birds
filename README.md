This project implements many of design patterns.

# Features


**Game Features:**


- Shooting bullets
- Five Levels
- Background music
- Sound effects
- New font
- Score chickens and kills
- Different bullets
- Different chickens
- Animated airplane
- 5 Health



**Design Patterns:**


- Decorator pattern
- Observer pattern
- Adapter pattern
- Singleton pattern
- Composite pattern
- Strategy pattern
- Template pattern



**IMPORTANT: This project is not uploaded completely , because it is not open source , it is just to CV preview.**
